Toto je malý kompiler pro jazyk podobný jazyku C.
(Jazyk je popsaný podrobně v souboru `dokumentace.pdf`.)

Přehled souborů:
	- `src/` - zdrojový kód
	- `makefile` - makefile pro kompilaci s programem `make`
	- `examples/` - příklady použití jazyka
	- `tests/` - testy souboru `src/lib/bignum.c`
	- `dokumentace.pdf` - dokumentace k projektu
	- `doc/` - LaTeX kód dokumentace
